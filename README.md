## 🌟 What is KamiBot?

KamiBot is a powerful Instagram growth automation tool built and branded by **Kamiklankreatives LLC**. Originally forked from taktik-bot, KamiBot has been customized for the **music, hip-hop, and creative arts niche** — helping independent artists, producers, and labels grow their Instagram presence without spending on ads.

---

## ⚡ Features

- 🎯 **Smart Targeting** — Follow users by hashtag, location, or competitor followers
- 🧠 **Natural Behavior Simulation** — Human-like delays, randomized actions to avoid bans
- 🎤 **Music Niche Defaults** — Pre-configured for hip-hop, rap, beats, and creative arts hashtags
- 🔄 **Auto Follow / Unfollow** — Grow and clean your following list automatically
- 📸 **Story Viewing** — Engage with stories to boost visibility
- 💬 **DM Bridge** — Extract and manage Instagram DM conversations
- 📊 **Analytics Logging** — Track actions, targets, and growth over time
- 🛡️ **Safe Mode** — Configurable rate limits to stay within Instagram's guidelines

## 🚀 Quick Start

### Requirements
- Python 3.10+
- Android device or emulator (for automation)
- ADB installed

### Installation

```bash
git clone https://github.com/odoublexisthebest-droid/kamibot.git
cd kamibot
pip install -r requirements.txt
python setup.py install
```

### Configure

Edit `taktik/config.py` with your Instagram credentials and targeting preferences.

### Run

```bash
python -m taktik
```
